# investimento_flutter_2026

Projeto simples desenvolvido em Flutter para estudo e prática de desenvolvimento mobile.

## Como o projeto funciona

Este projeto foi criado com Flutter e utiliza Dart como linguagem principal.

A estrutura principal funciona assim:

- `lib/main.dart`: arquivo principal da aplicação, onde o app inicia.
- `assets/`: pasta usada para arquivos visuais, como imagens.
- `android/`, `ios/`, `web/`, `windows/`, `linux/`, `macos/`: pastas de configuração e compilação para cada plataforma suportada pelo Flutter.

Quando o projeto é executado, o Flutter inicia a aplicação a partir do arquivo `main.dart`, renderizando a interface na plataforma escolhida, como Android, Windows ou Web.

## Tecnologias usadas

- **VS Code**: editor utilizado para desenvolver o projeto.
- **Flutter**: framework usado para criar a aplicação.
- **Extensão Flutter**: extensão do VS Code que facilita executar, depurar e gerenciar o projeto Flutter.
- **Extensão Dart**: extensão do VS Code necessária para suporte à linguagem Dart.

## Como executar o projeto

1. Instale o **Flutter SDK**.
2. Abra o projeto no **VS Code**.
3. Instale as extensões **Flutter** e **Dart** no VS Code.
4. Execute o comando para caso a imagem de fundo não apareça:

```bash
flutter pub get
```

5. Depois, rode o projeto com:

```bash
flutter run
```

## Objetivo

O objetivo deste projeto é servir como base para aprendizado e evolução de uma aplicação Flutter multiplataforma.
